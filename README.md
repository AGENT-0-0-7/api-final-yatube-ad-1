# API проекта Yatube

## Описание
Проект Yatube API — это RESTful API для социальной сети Yatube. Он позволяет взаимодействовать с платформой программным путем: создавать и редактировать публикации, оставлять комментарии, подписываться на авторов и просматривать тематические сообщества. 
API предоставляет доступ только для чтения неавторизованным пользователям, в то время как авторизованные получают полный функционал управления своим контентом.

## Стек технологий
* Python
* Django, Django REST Framework
* Djoser (JWT Authentication)
* SQLite3
